# Crowd Panic Detection System

An AI-based application that detects panic situations in crowded environments using computer vision and deep learning techniques.

---

## 🚀 Features
- Real-time video analysis using CCTV footage
- Person detection using YOLOv8
- Emotion detection using DeepFace
- Pose estimation using MediaPipe
- Panic score calculation based on behavior
- Automatic alert generation when panic is detected

---

## 🛠 Tech Stack
- Python
- OpenCV
- YOLOv8
- DeepFace
- MediaPipe

---

## ⚙️ How It Works
1. Capture video frames from CCTV or input video
2. Detect people using YOLOv8
3. Analyze emotions (fear, stress) using DeepFace
4. Analyze body posture using MediaPipe
5. Calculate panic score
6. Trigger alert if panic score exceeds threshold
